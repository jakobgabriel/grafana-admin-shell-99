About Grafana Dashboard Explorer

A powerful tool for managing and comparing Grafana dashboards across multiple instances

Overview

The Grafana Dashboard Explorer helps teams manage and monitor their Grafana dashboards across different environments. It provides insights into dashboard deployment status, statistics, and quick access to all dashboard links in one place.

Features

  • Compare dashboard deployments across instances
  • View comprehensive statistics and metrics
  • Access all dashboard links in one centralized location
  • Monitor folder and dashboard counts across instances
